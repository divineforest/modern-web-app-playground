# AI Friction Check

Evaluate the task you just completed:

1. **Friction**: What slowed you down or was difficult? (Be specific - file names, missing context, confusing patterns)
2. **Suggestion**: One concrete improvement to reduce that friction
3. **Score**: Rate suggestion 1-10 (10 = high confidence it actually helps AI, not just "seems useful")

Rules:
- Only report friction you actually experienced
- If task was smooth, just say "No friction" and skip the rest
- Be skeptical of documentation-based solutions - code quality usually matters more
- Score < 7 = probably not worth implementing
